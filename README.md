# OpenFishbanks

OpenFishbanks is a free and open-source web-based fishing economy simulator inspired by MIT Sloan's Fishbanks. The simulation can be run over the course of a week or more and requires multiple users to participate. Data such as fish species population, revenue, value of fish, value of boats, and stock of fish and boats is automatically stored in the db for later analysis.
It is built in Django and uses Celery to asynchronously update fish populations and user revenues, so users do not need to be consistently logged in to fish.
